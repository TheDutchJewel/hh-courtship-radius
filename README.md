# Courtship radius for webtrees

`hh-courtship-radius` is a webtrees 2.2 custom module for analysing the geographic courtship radius of families selected through the tree-specific clippings cart.

The planned analysis groups marriages into time slices, calculates separate courtship radii for men and women, reports data quality, and provides statistical, cross-table, chart, map, and CSV views.

## Status

The module is under initial development. Version 0.1.0 contains the initialized project structure; functional implementation follows in subsequent commits.

## Requirements

- webtrees 2.2
- the webtrees clippings-cart module for selecting individuals and families
